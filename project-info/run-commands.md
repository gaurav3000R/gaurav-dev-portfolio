# Run Commands

This project uses `npm` as the package manager. Here are the most common commands, defined in the `scripts` section of the `package.json` file.

-   **`npm run dev`**
    -   Starts the development server.
    -   The site will be available at [http://localhost:3000](http://localhost:3000).

-   **`npm run build`**
    -   Builds the application for production.
    -   The output is stored in the `.next` directory.

-   **`npm run start`**
    -   Starts the production server.
    -   You need to run `npm run build` before using this command.

-   **`npm run lint`**
    -   Runs the ESLint linter to check for code quality and style issues.
